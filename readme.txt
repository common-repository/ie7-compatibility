Enables Internet Explorer 7 (IE7) compatibility until your blog works with IE8, Microsoft's new browser.

== Description ==

Internet Explorer 8 (IE8) is coming up, is your site ready?  IE7 compatibility can help if your site needs some fixes done to be ready.  When activated, your site will look normal to all visitors.  Once you have fixed the problems, just deactivate or delete the plugin.

== Installation ==

This section describes how to install the plugin and get it working.
